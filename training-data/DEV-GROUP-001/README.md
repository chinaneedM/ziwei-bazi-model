# DEV-GROUP-001

This immutable development group contains five reusable cases and 25 questions. Runtime objects contain no answer payload. Cases 001-004 are stored as gzip+base64 JSON and case 005 as JSON; `scripts/materialize-training-case.py` reads either format. Private answers live only in `chinaneedM/fortune-answer-vault`. All five baseline predictions must freeze before reveal authorization. Future training examples use a new immutable group or revision and are never silently overwritten.

Bazi screenshot binaries are represented by verified structured transcription plus original SHA-256, dimensions, byte size, and ChatGPT file ID; raw PNG bytes are not committed through the current connector.
