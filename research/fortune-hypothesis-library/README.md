# Research Hypothesis Library

This directory stores project-discovered reasoning hypotheses separately from the installed S00-S19 baseline.

Knowledge states:

- `SOURCE_CONFIRMED_KNOWLEDGE`: source-backed, installed and machine-validated.
- `REASONED_HYPOTHESIS`: a conditional interpretation discovered during review or cross-case analysis.
- `OPEN_RESEARCH_QUESTION`: plausible but not yet sufficiently specified or tested.
- `REJECTED` or `SUPERSEDED`: retained for audit but never activated.

Promotion path:

`PROPOSED -> SPECIFIED -> RETROSPECTIVE_SUPPORT_ONLY -> SAME_SET_CLEAN_SUPPORT -> UNSEEN_VALIDATION_ACTIVE -> VALIDATED_CANDIDATE -> PROMOTED_TO_ACTIVE_SOURCE`

A single case or post-reveal explanation cannot promote a hypothesis. Promotion requires a general statement, provenance, mechanism, conditions, limitations, counterexamples, clean same-set support, independent unseen validation, contradiction review, user approval for an S00-S19 change, and complete binding validation.

Same-set clean replay tests whether a lesson was absorbed. New unseen cases test whether it generalizes.
