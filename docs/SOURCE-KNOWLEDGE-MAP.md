# S00–S19来源库系统梳理

## 结论

这套来源库不应被当作二十本可以同时“投票”的书，而应被当作一条分层推理链。S04–S10负责紫微坐标、静态结构、流派技法和岁运，S11–S16负责八字结构与岁运，S02/S04/S17把命理象义闭合为现实人物、动作和结果，S03/S18完成双轨裁决与选项比较，S00/S01/S19负责检索、输入和运行边界。

“南派看静态、北派看动态”可以作为入门记忆，但不能成为系统中的硬分界。中州星系也使用四化，飞星也离不开本命星情和宫位；河洛的体用、气数位与天地人是另一套坐标和层次工具。不同流派必须先分别说明自己在当前题目中能证明到什么程度，再去重、比较和融合，不能按流派数量表决。

## 二十库职责

| 库 | 核心职责 | 运行时回答的问题 | 禁止越级 |
|---|---|---|---|
| S00 | 全库索引、来源定位、版本与派生索引边界 | 该去哪里找、读取哪个活动父段 | 索引摘要不能代替正文或贡献命理事实 |
| S01 | 输入冻结、历法/盘面坐标、转录与答案隔离 | 输入究竟是什么、哪些字段可靠 | 模糊字段不得猜造；答案不得进入预测快照 |
| S02 | 现代语义、地域时代、量级、状态和参考期 | 选项在现实中要求证明哪些原子 | 现代定义本身不能证明事件发生 |
| S03 | 紫微与八字独立轨的冲突裁决 | 两轨同向、互补、冲突还是一轨无能力 | 不能复制另一轨的理由，也不能按票数融合 |
| S04 | 十二宫主题、实体/关系拓扑、主题太极与气数位入口 | 问谁、问什么、宫位和人物入口在哪里 | 公司、钱、关系、事件不是自然人，不能伪造人物太极 |
| S05 | 十四主星、辅煞杂曜、组合条件与格局基础 | 星曜及组合的稳定性质、条件和限制 | 单星不能广播到性格、婚姻、职业、健康全部领域 |
| S06 | 十二基础盘和六十星系物理结构 | 当前主星组合与三方四正构成什么完整星系 | 星系结构不能直接替代入宫语义或现实结果 |
| S07 | 星曜/星系进入十二宫后的条件化表达 | 同一星系在具体宫位如何表现 | 入宫倾向不能直接完成现代职业、诊断或法律终点 |
| S08 | 十干四化、生年四化、自化、向心/离心与禄忌线 | 哪个宫位/人物/关系被何种动力连接和改变 | 动态显著不等于已经发生某个精确事件 |
| S09 | 一四四诀、体用飞化、主题太极检索 | 从选定主题的十二角度查找体、用和条件链 | 诀条是条件字典，不是命中即断的答案表 |
| S10 | 本命、大限、流年、流月与动态辅助 | 哪一层提供结构，哪一层许可/触发，何时转折 | 先判中性应期，再判吉凶、主体与终点 |
| S11 | 八字干支、五行、藏干与十神 | 四柱的原始角色和十神关系是什么 | 十神人物不是现实人物的一一映射 |
| S12 | 月令、旺衰、通根与气势 | 日主和全局在季节中如何承载、流通和从化 | 不以简单身强身弱覆盖全局结构 |
| S13 | 格局、用忌、调候与病药 | 哪种解释轴能说明结构功能与失衡 | 格局、扶抑、调候、病药可并列，不能强造唯一用神 |
| S14 | 合冲刑害、墓库、伏吟反吟与结构变化 | 什么关系被合化、冲开、重复或改变 | 一个冲、伏吟或墓库只能给变化候选，不能指定精确事件 |
| S15 | 大运、流年、流月与八字应期 | 结构在何运何年获得许可、加强或转向 | 年份显眼与题目事件能力必须分开 |
| S16 | 八字专题映射和紫微接口 | 八字对婚姻、职业、财富、健康等能证明到哪一级 | 缺主体、动作、手续或终点时应局部弃权，不复制紫微 |
| S17 | 人物太极、现实动作链与终点闭合 | 倾向是否到机会、发生和正式终点 | 风险或机会不能跳成死亡、结婚、职业、金额等完成态 |
| S18 | 完整证据账、同源去重、复合覆盖与成对选择 | 第一为何胜第二、最强竞争项能否反转 | 不固定“2紫微+1八字”；不以证据数量或流派数量投票 |
| S19 | 运行、冻结、揭盲、训练与审计边界 | 何时可预测、评分、学习和发布 | 不保存答案映射，不允许同题回放充当独立验证 |

## 紫微各流派的接口

### 《紫微斗数全书》与传统静态体系

核心价值在于星曜本性、十二宫表达、辅煞组合、三方四正与格局条件。它回答“先天结构像什么”，适合建立本命底盘和稳定倾向。传统断语必须连同庙旺落陷、会照、煞曜、性别、昼夜、组合条件和异文一起读取，不能把一句古诀当作现代事实必然律。

### 中州派与六十星系

中州方法的重点不是逐颗星累加，而是把主星组合视为完整星系，再结合三方四正、辅佐煞曜和四化判断星系性质如何改变。六十星系是结构索引，不是六十条结果标签。中州四化主要用来修正星曜与星系性质，并且库中明确存在流派异表；必须先锁定同一套十干四化表，不能把《全书》表、中州表和飞星表混用。正确顺序是：确认物理星系（S06）→读取星系本性与组合条件（S05）→投影到宫位（S07）→以同一流派四化表修正（S08）→再看岁运（S10）。

### 梁若瑜飞星派

飞星的长处在于描述动态关系：十干引发四化，生年四化表达先天动力，自化反映宫位内部的释放、转向或不稳定，禄忌线表达宫与宫、人与人、事与事之间的作用路径。它适合回答“哪一层被触发、谁向谁作用、关系如何变化”，但仍需S04确定人物和主题、S17完成现实动作与结果。

### 河洛派、方外人、气数位与天地人

河洛体系用体、用、宫干、气数位和天地人层次建立另一套观察坐标。体是当前问题的立足点，用是所问对象或作用方向；气数位在本库中是主题宫于该主题太极中的官禄位，表示现实承接，不是自动成功或失败宫。方外人“天地人三盘”在本库的具体时间接口是本命—大限—流年：大限作为媒介上应本命、下应流年，四化沿层级作用并回看本命；不能把它泛化成任意三层。它们最适合作为“重新立太极后从哪个角度查”，不能替代人物入口、时间激活或终点链。

### 一四四诀

一四四诀可以理解成一套体用条件字典，但它的144不是“12个主题×12个角度”，而是**12体宫×12用宫**的144个体用诀；每诀再展开12个太极点和四化节点。运行时应先确定题目主语与主题，建立`BODY_PALACE`、`USE_PALACE`、`USE_GAN`、`TAIJI_POINT`和`TRANSFORMATION`，再查对应节点的条件、否定、归零标签和能力上限，最后回到实际盘面、动态层与现实终点。单条诀不能直接定事。

## 紫微时间层

1. 本命原盘：决定可承载的稳定结构和主题底色。
2. 大限：决定十年舞台、主题是否进入主要活动区以及结构如何重排。
3. 流年：决定具体年份触发、动作方向和转折窗口。
4. 流月：只在题目需要更细时间且上层已成立时细化。

“本命有象”不等于一生必发生；“流年很动”也不等于选项中的精确事件。正式推理必须把原盘结构、大限许可、流年触发和现实终点接成同一条链。

## 八字系统骨架

八字不从“看见十恶大败、伏吟或一个冲”开始，而从以下顺序开始：

1. 冻结节气四柱、性别、出生时间、起运和大运。
2. 建立干支五行、藏干和十神，但十神只作角色候选。
3. 以月令、通根、透干、组合和全局气势判断承载方式。
4. 分别检查格局、扶抑、调候、病药和流通，不强迫它们合成一个万能“喜用神”。
5. 用合冲刑害、墓库、伏吟反吟描述结构如何变化。
6. 进入大运、流年、流月，先判断中性激活，再判断方向。
7. 经S16把结构能力映射到题目专题，经S17闭合现实人物、动作和终点。

伏吟是干支/宫位或岁运结构的重复放大，可能表示重复、停滞、回返或同类主题增强；它不是“必凶事件标签”。十恶大败等神煞只能作为低权重辅助或检索提示，不能单独决定财富、死亡、婚姻或事故。

## 双轨融合

紫微和八字必须先独立完成。融合时只允许三种主要结果：

- 同向但同源：只算一次物理事实，不叠加权重。
- 真正互补：一轨补齐另一轨缺失的主体、时间、方向或强竞争反证。
- 冲突/能力不对称：按题目轴、来源条件、时间命中和终点闭合度裁决；能力不足的一轨可以局部弃权。

两轨结论相同不自动增加可信度；两轨结论不同也不允许简单平均。

## 与训练模型的关系

来源库是冻结知识底座，不因一道题对错而修改。揭盲后的学习只允许进入`model-learning/`，而且只能修正知识调用顺序、证据权重、适用边界、反证程序和能力上限。案例专属答案、题号、选项字母或原句不得进入通用规则。

S19现存文本含“同案连续三轮”门槛；当前有效R1已在`config/training-policy.json`和控制器中恢复该门槛。V3的题级首次盲测、三分区与未来不同案例验证规则继续作为诊断和证据层，但不得覆盖R1推进门。冻结S19不改写，政策关系通过派生审计显式记录。
