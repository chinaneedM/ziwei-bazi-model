## 长期评估

模型不使用一个总分宣布“学会”。长期同时报告首次盲测Top1、Top2、主题/人物/终点/推理能力分层结果、置信度校准、每条规则的不同案例支持与反例、阶段验证退化、最终保留成绩和样本不足领域。

每次错误只允许产生候选规则。规则发布前必须做适用范围检查和回归检查；阶段验证不能当场造规则，最终保留只用于阶段性结论。新增数据优先补稀缺主题、相似结构的负例和现实基础率，而不是只收集戏剧性命中案例。

具体题级分布见`docs/CASE-COVERAGE-REPORT.md`。
