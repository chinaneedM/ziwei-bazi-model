## 二、流派关系的系统理解

“南派看静态、北派看动态”可以用于入门，但不能作为硬分界。

- 传统静态体系先建立星曜本性、辅煞组合、三方四正和格局条件。
- 中州派把完整星系作为单位，四化用于修正星系性质，而不是逐星累加。
- 梁派飞星强调宫干四化、自化、向心/离心和禄忌路径，擅长描述动力方向与关系变化。
- 河洛/方外人以体用、气数位和天地人重建观察坐标；气数位是现实承接位置，不是自动吉凶宫。
- 一四四诀是12体宫×12用宫的条件字典，再按用干、太极点和四化精确查找；不是“命中一句即断”。
- 本命提供结构，大限提供十年舞台和媒介，流年提供具体触发，流月只在上层成立且题目需要时细化。

系统不按流派数量投票。每一派先声明自己能证明到哪一级，再按来源独立性、时间命中和现实终点去重、比较和融合。

## 三、八字工作流

八字不得从神煞、伏吟或一个冲开始。固定工作流为：

1. 冻结节气四柱、性别、出生时间、起运和大运。
2. 建立日主、干支五行、藏干与十神；十神只提供角色候选。
3. 以月令、通根、透干、组合、寒暖燥湿和全局流通建立承载候选。
4. 让格局、扶抑、调候、通关、病药和从化并行竞争，不强造唯一喜用神。
5. 用合冲刑害、墓库、伏吟反吟描述结构如何变化，区分关系存在与转化成立。
6. 加入大运、流年和必要流月，先判中性许可与激活，再判方向。
7. 经专题映射判断八字对人物、职业、婚姻、财富、健康等能证明到哪一级。
8. 缺人物、动作、手续、持续或终点时局部弃权，不复制紫微结论。

十恶大败等神煞只作低权重检索提示；伏吟可以是重复、停滞、回返或增强，不能自动等于凶事。

## 四、R2泛化训练模型

“一案一轮后永不复训”覆盖广但不能验证修复；“同案连续三轮”又容易把答案记忆误当成能力。R2把两者拆成首次盲测与间隔复训两条指标：

- 每轮答完当前案例全部题；
- 少于5题全对，5题及以上达到向上取整80%；
- 每个新案只计一次首次盲测，闭环后进入下一新案；
- 新案失败立即把跨案连续次数归零，并要求通用复盘和`model-learning`修正；
- 失败案进入复训队列，至少隔开5个新案后才可复训；
- 同案复训只验证修复，不计首次盲测、独立规则证据或阶段门；
- 3个不同首次盲测案例连续达标才通过阶段泛化门；
- 65例开发集可学习，21例阶段验证集和21例最终留出集不得当场造规则。

历史状态按事实迁移：`DEV-EXAMPLE-001`的`ROUND-001`是首次盲测失败，`ROUND-002`是同案复训通过；因此修复状态已关闭，但跨案连续达标数仍是0。整个107例数据集仍保持`DATASET_FROZEN_AWAITING_ANSWER_IMPORT`，不会开放新预测。

## 五、人类经验与AI都需要防范的盲区

用户经验进入`USER_HYPOTHESIS`，AI补充进入`AI_HYPOTHESIS`；两者使用同一证据门槛。

必须长期防范：

- 确认偏误：只记命中，不记录相似结构但未发生；
- 教学例题选择偏差：题库偏重大事件，不能代表现实基础发生率；
- 结果泄漏：来源命例结局、旧预测和答案不得回流未揭盲推理；
- 流派混表：不同四化表、用神体系和术语不得静默拼接；
- 时代地域错配：职业、住房、财富、教育和医疗量级必须按当时当地解释；
- 答案权威偏误：所谓官方答案也应记录来源、置信度和争议状态；
- 叙事闭合偏误：故事顺畅不等于人物、动作、时间和终点已经闭合；
- 样本不足：灵异、性取向、死亡、赌博横财等主题不得高确定性泛化；
- 置信度失真：长期比较70%、80%等自报置信度与真实命中率；
- 模型自证：测试通过只证明工程链，不证明命理预测有效。
