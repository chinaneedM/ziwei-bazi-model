## 由覆盖矩阵得出的结论

- 婚姻、职业、健康、财富、父母和年份事件已有小规模跨分区验证条件。
- 灵异/宗教类全部在开发区，只能提出假设，不能声称获得独立验证。
- 性取向没有阶段验证样本，最终保留也只有1题；不得从这些例题形成确定断法。
- 迁移与兄弟主题没有最终保留覆盖，赌博横财也只有极少阶段/保留样本；成绩必须显示“样本不足”。
- 教学题显著偏向重大年份、职业和戏剧性事件，不能把题库频率当作现实世界基础发生率。
- 后续新增案例应优先补灵异主张的可证伪对照、性取向的伦理边界、迁移、兄弟、普通未发生事件和与高风险盘象相似但没有出事的负例，而不是继续堆积重大事件题。
