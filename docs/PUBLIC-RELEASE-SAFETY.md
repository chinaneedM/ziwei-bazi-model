# 公共仓库发布说明

## 授权与范围

资料提供者于2026-07-23明确选择继续使用公共仓库，并说明来源库内容为其个人笔记与整合材料，同意本项目以纯开源方式发布。本记录用于消除此前“是否知情同意公开”的不确定性，不构成对第三方权利或任何司法辖区法律结论的替代。

## 发布前检查

- 案例运行目录未发现姓名、身份证/身分证、护照号、电话、微信、邮箱或详细住址标签。
- 未发现中国大陆手机号、居民身份证号、电子邮箱或常见GitHub/云服务密钥格式。
- 未发现正确答案、评分、揭盲复盘、旧预测或明文答案密钥。
- `answer-vault/`只允许加密信封；密钥只存在外部GitHub Actions Secret。
- 例题仍包含出生日期、出生时间、出生地、命盘和人生事件。这些是敏感的准识别信息，虽无姓名仍可能在与外部资料组合后识别人物。

## 固定发布边界

1. 不把原始答案包、解密密钥、旧预测、逐题评分或复盘映射提交到公共仓库。
2. 新增案例在发布前继续扫描直接身份信息和答案污染。
3. 如果未来材料出现姓名、联系方式、证件号、详细地址或未经授权的第三人资料，默认脱敏或阻塞，不因本次授权自动公开。
4. 来源归属或第三方权利出现具体争议时，先定位相关文件和段落，再决定标注、替换或移除；不静默改写冻结原典。
