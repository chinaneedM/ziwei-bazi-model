# Group training user command

The canonical user start command is:

> 继续仓库固定五案例的整组盲态循环训练。

After this single instruction, the active CHAT/WORK session is expected to:

1. load the frozen answer-free group manifest;
2. process all cases in order with fresh per-case isolation;
3. freeze the complete group;
4. authorize group reveal only after freeze;
5. perform diagnosis, candidate repair, clean rerun, regression comparison, and accept or rollback according to policy.

The executor must not ask the user to open a new conversation or send a separate continue instruction for each case.
